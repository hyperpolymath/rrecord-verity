/* SITE */

Project: DKIM Verifier
Description: Email authentication and security extension for Mozilla Thunderbird
Repository: https://github.com/lieser/dkim_verifier
Wiki: https://github.com/lieser/dkim_verifier/wiki
Add-ons Page: https://addons.thunderbird.net/thunderbird/addon/dkim-verifier/
Standards: DKIM (RFC 6376), SPF (RFC 7208), DMARC (RFC 7489)
License: MIT
Last Update: 2025-11-22
Language: JavaScript (ES2024) with TypeScript checking
Version: 7.0.0-beta

/* TECHNOLOGY */

Backend: JavaScript ES2024, WebExtension APIs
Type System: TypeScript via JSDoc (@ts-check)
Crypto: tweetnacl-es6 (Ed25519, RSA via Web Crypto API)
DNS: JSDNS (JavaScript) + libunbound (native)
UI: HTML5, CSS3, Tabulator Tables
Testing: Mocha, Chai, Sinon
Build: Node.js, ESLint, package.json scripts
CI/CD: GitHub Actions
Platform: Mozilla Thunderbird 128+
